Project Title: Full-Stack Financial & Task Management Dashboard
Overview
Designed and engineered a complete, end-to-end CRUD (Create, Read, Update, Delete) application to help users track active tasks, incoming revenue, and business expenses. I architected the entire system from the relational database and REST API backend up to the responsive, dynamic frontend.

Core Features

Real-Time Data Processing: Engineered a RESTful API to handle asynchronous GET and POST requests, allowing the frontend to update seamlessly without page reloads.

Persistent Data Storage: Configured an embedded relational database with Hibernate ORM to ensure data permanence and integrity across server restarts.

Cross-Origin Resource Sharing (CORS): Implemented secure routing to allow the distinct frontend and backend servers to communicate flawlessly.

Modern UI/UX: Built a responsive, dark-mode visual interface using modern React hooks (useState, useEffect) for optimal state management and user experience.

Technical Stack

Frontend: React, Vite, HTML5, CSS3, JavaScript (ES6+)

Backend: Java 21, Spring Boot, Spring Web

Database & ORM: H2 Database, Spring Data JPA, Hibernate

Tools: Git, GitHub, Postman API Platform, IntelliJ IDEA, Node.js
